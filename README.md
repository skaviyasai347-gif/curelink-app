# 📱 CureLink Mobile App – Flutter (Dart)

CureLink Mobile App is a modern **healthcare application built using Flutter (Dart)**, designed to provide a smooth and accessible healthcare experience directly from mobile devices.
This app complements the CureLink platform by offering a fast, responsive, and user-friendly interface.

---

## 🚀 Project Overview

The mobile application focuses on delivering healthcare services in a simple and efficient way.

* 📱 Cross-platform mobile app (Android & iOS)
* ⚡ Fast and smooth performance using Flutter
* 🎯 User-centric design for easy navigation
* 🏥 Designed for digital healthcare accessibility

---

## 🛠️ Tech Stack

* Flutter
* Dart

* 🩺 Clean and intuitive healthcare UI
* 📋 Easy navigation between screens
